# Lung Cancer Detection Module

## Overview
This module detects Lung Cancer from CT scan images using
radiomics feature extraction and classical machine learning.

## Dataset
- Normal
- Benign → Cancer
- Malignant → Cancer

## Pipeline
Image → Feature Extraction → Scaling → Random Forest → Prediction

## Features Used
- Homogeneity
- Dissimilarity
- Contrast
- Correlation
- Mean Intensity
- ASM
- Energy
- Std Intensity

## Model
- Random Forest Classifier
- Balanced training data
- Accuracy ~96%

## Disclaimer
This AI system is for educational and research purposes only.
It is not a replacement for professional medical diagnosis.
